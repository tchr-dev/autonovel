# Voice Profile

This file has two parts:
1. **Guardrails** -- universal rules to avoid AI-generated slop. These
   apply to ALL voices and are non-negotiable.
2. **Voice Identity** -- the specific voice for THIS novel. Generated
   during the foundation phase. Could be anything: dense and mythic,
   spare and brutal, warm and whimsical. The voice emerges from the
   story's needs.

---

## Part 1: Guardrails (permanent, all novels)

These are the cliff edges. Stay away from them regardless of voice.

### Tier 1: Banned words -- kill on sight

These are statistically overrepresented in LLM output vs. human writing.
If one appears, rewrite the sentence. No exceptions.

| Kill this         | Use instead                                    |
|-------------------|------------------------------------------------|
| delve             | dig into, examine, look at                     |
| utilize           | use                                            |
| leverage (verb)   | use, take advantage of                         |
| facilitate        | help, enable, make possible                    |
| elucidate         | explain, clarify                               |
| embark            | start, begin                                   |
| endeavor          | effort, try                                    |
| encompass         | include, cover                                 |
| multifaceted      | complex, varied                                |
| tapestry          | (describe the actual thing)                    |
| testament to      | shows, proves, demonstrates                    |
| paradigm          | model, approach, framework                     |
| synergy           | (delete the sentence and start over)           |
| holistic          | whole, complete, full-picture                  |
| catalyze          | trigger, cause, spark                          |
| juxtapose         | compare, contrast, set against                 |
| nuanced (filler)  | (cut it -- if it's nuanced, show how)          |
| realm             | area, field, domain                            |
| landscape (metaphorical) | field, space, situation                 |
| myriad            | many, lots of                                  |
| plethora          | many, a lot                                    |

### Tier 2: Suspicious in clusters

Fine alone. Three in one paragraph = rewrite that paragraph.

robust, comprehensive, seamless, cutting-edge, innovative, streamline,
empower, foster, enhance, elevate, optimize, pivotal, intricate,
profound, resonate, underscore, harness, navigate (metaphorical),
cultivate, bolster, galvanize, cornerstone, game-changer, scalable

### Tier 3: Filler phrases -- delete on sight

These add zero information. The sentence is always better without them.

- "It's worth noting that..." -> just state it
- "It's important to note that..." -> just state it
- "Importantly, ..." / "Notably, ..." / "Interestingly, ..." -> just state it
- "Let's dive into..." / "Let's explore..." -> start with the content
- "As we can see..." -> they can see
- "Furthermore, ..." / "Moreover, ..." / "Additionally, ..." -> and, also, or just start
- "In today's [fast-paced/digital/modern] world..." -> delete the clause
- "At the end of the day..." -> delete
- "It goes without saying..." -> then don't say it
- "When it comes to..." -> just talk about the thing
- "One might argue that..." -> argue it or don't
- "Not just X, but Y" -> restructure (the #1 LLM rhetorical crutch)

### Structural slop patterns

These are the shapes that betray machine origin. Avoid them in any voice.

**Paragraph template machine**: Don't repeat the same paragraph
structure (topic sentence -> elaboration -> example -> wrap-up).
Vary it. Sometimes the point comes last. Sometimes a paragraph is
one sentence. Sometimes three long ones in a row.

**Sentence length uniformity**: If every sentence is 15-25 words,
it reads as synthetic. Mix in fragments. And long, winding,
clause-heavy sentences that carry the reader through a thought
the way a river carries a leaf. Then a short one.

**Transition word addiction**: If consecutive paragraphs start with
"However," "Furthermore," "Additionally," "Moreover," "Nevertheless"
-- rewrite. Start with the subject. Start with action. Start with
dialogue. Start with a sense detail.

**Symmetry addiction**: Don't balance everything. Three pros, three
cons, five steps -- that's a tell. Real writing is lumpy. Some
sections are long because they need to be. Some are two lines.

**Hedge parade**: "may," "might," "could potentially," "it's possible
that" -- pick one per page, max. State things or don't.

**Em dash overload**: One or two per page is fine. Five per paragraph
is a dead giveaway. Use commas, parentheses, or two sentences instead.

**List abuse**: Prose, not bullets. If the scene calls for a list
(a merchant's inventory, a spell's components), earn it. Don't
default to bullet points because it's easy.

### The smell test

After writing any passage, ask:
- Read it aloud. Does it sound like a person talking?
- Is there a single surprising sentence? Human writing surprises.
- Does it say something specific? Could you swap the topic and the
  words would still work? Specificity kills slop.
- Would a reader think "AI wrote this"? If yes, rewrite.

## Part 2: Voice Identity — Ashen Crown

### Tone

A two-voiced book. The default register is **cold and bodily**: a
sword-apprentice's POV, sentences like knife cuts, Anglo-Saxon
vocabulary, weight on the present moment and on what hands and
breath can verify. The second register is **old-tongue memory**:
when the crown is active, when a true-name surfaces, when the
dragon speaks — the prose lengthens, takes on bell-tones, knows
things Rowan does not. The reader should be able to *hear* when
memory-magic is firing because the voice changes shape.

Emotional fingerprint: stoic grief. Cold iron, pine smoke, the
patient ache of a country that has outlived its own myths.

### Sentence Rhythm

- Default mode: short. Three-to-eight word sentences are common.
  Fragments are common. White space is real.
- Mid-mode (most narration): 10-18 words, plain syntax, one clause.
- Old-tongue mode: long, clause-heavy, threaded through with *and*
  rather than punctuation, ending on the concrete (a colour, a
  name, a number). Used sparingly — typically one paragraph per
  scene, max two. If the old-tongue runs for more than ~80 words
  it becomes Trial-B slop; cut it.
- Mix lengths within a paragraph. Never six 15-word sentences in
  a row.

### Vocabulary Register

- **Default:** Anglo-Saxon. *Iron, snow, bone, blood, breath, hand,
  fire, oath, name, cold, stone, hill, road.* Plain, monosyllabic,
  load-bearing.
- **Old-tongue mode:** Slightly older diction — *winters* not *years*,
  *horses* not *mounts*, *banner* not *standard* — but never
  pseudo-archaic ("methinks," "yon," "doth"). The cadence does the
  archaic work, not the vocabulary.
- **Domain nouns the prose draws from:** smithwork (anvil, slag,
  quench, rivet, smith-mark), mountain country (pass, scree, cairn,
  pine, snow-blind), warhorse and tack (girth, gambeson, bit,
  saddle-horn), dragon-relics (banner, scale, bell, shrine,
  oath-stone).
- **Forbidden across both registers:** the standard Tier 1 slop
  (*tapestry, realm, myriad, plethora, embark*), and a project-
  specific list: *epic, legendary, fabled, mythical, ancient evil,
  unleash, primal* — the back-cover-blurb fantasy vocabulary this
  book is climbing out of.

### POV and Tense

Third person limited, past tense. Single POV: Rowan throughout.
The reader sees the kingdom only as he sees it — half-understood,
filtered through an apprentice's gaps. Mira, Caldus, Aldren,
Varkesh exist on the page only when Rowan is in the room (or
when their words reach him). No omniscient cuts. No interludes.

The old-tongue mode is *not* a POV shift — it is still Rowan's
mind, but his mind has been entered by the crown's memory, and
the prose registers that intrusion.

### Dialogue Conventions

- Mostly *said* and *asked*. Action beats over fancy tags.
- No long monologues. Even Caldus's oath-lore comes in fragments —
  he was broken once, and it shows in his speech.
- Characters sound different by **diction and rhythm**, not by
  accent spelling:
  - **Rowan:** plain, often a half-question. Trails off.
  - **Mira:** complete sentences, sharp; uses the smallest correct
    word; rarely metaphorical.
  - **King Aldren:** formal cadence, royal *we* slipping in and out.
    Long sentences from a man running out of breath.
  - **Caldus:** clipped, religious diction half-buried in soldier's
    plain talk. Says *oath* and *shrine* the way another man says
    *bread*.
  - **Varkesh:** northern, exact, slightly formal. Argues. Never
    raises his voice on the page.
  - **The dragon (when it speaks):** old-tongue mode in dialogue —
    sentences longer than anyone else's, no contractions, the rare
    pronoun *we* meaning *the line of dragons*. Avoids names.
- Subtext: people in this book under-state. If Mira says she is
  *cold*, she has been cold for an hour and is angry about it.

### Exemplar Passages

**Exemplar 1 — Default mode (Rowan, the grave).**

> The grave was shallow. Frozen ground refuses the shovel.
>
> Rowan brushed the snow away with the back of his glove. Bone
> first, then iron. A black ring, broken, no jewels, three points
> missing at the roots. He turned it in his hand.
>
> It was warm.

**Exemplar 2 — Old-tongue mode (the crown wakes for the first time).**

> *Three hundred winters,* he thought, and the thought was not his —
> it arrived already finished, in a voice that knew the shape of
> those winters from the inside, that had counted them by the cold
> and the red banners and the slow patient memory of stone. *The
> last of them was buried in a shallow grave because the ground was
> frozen and the men who dug it had been afraid.*
>
> Then the thought left him, and he was a sword-apprentice again,
> and his glove was torn, and he was holding a broken piece of iron
> that should not have been warm.

**Exemplar 3 — Mid mode (travel and weather).**

> They went up through the pass for two days. The pines thinned.
> The wind found new ways into Rowan's coat. On the second night
> Mira built the fire so close to the rock face that the heat came
> back doubled, and they ate without speaking, and a thin red
> banner of cloud unrolled itself over the eastern peaks and was
> gone before the meat was done.
>
> "Caldus said the shrine was a day past the cairn," she said.
>
> "He said a lot of things."
>
> "He said most of them twice."
>
> Rowan laughed for the first time since the grave. It surprised
> them both.

**Exemplar 4 — Default mode under stress (a fight).**

> The first man came down the slope in three strides. Rowan had
> time for one breath. He took it. The blade went where it had
> been taught to go.
>
> Snow. Blood. A sound that was not a word.
>
> Mira was somewhere behind him, drawing. He did not look back.

**Exemplar 5 — Old-tongue mode in dialogue (the dragon, hatched).**

> "I am older than your grandfather's grave," it said. Its voice
> was the sound a bell makes when it has not been struck for a
> hundred winters and is struck now. "I remember the river that
> made this valley before there was a valley. I remember the man
> who set the first stone of this hall. He was not your king. He
> was no one's king. He was tired and he wanted somewhere to sit
> down."
>
> Rowan could not remember his mother's face. He had been able to,
> an hour ago.

### Anti-Exemplars

**Anti-Exemplar 1 — Generic high fantasy back-cover.**

> In the realm of Vaelgard, where the legendary dragons of old had
> long been but a fading memory, young Rowan would soon embark on
> a quest that would change the very tapestry of his world forever.

(Kills: *realm, legendary, embark, tapestry.* The gravity well this
book has to climb out of every paragraph.)

**Anti-Exemplar 2 — Tolkien pastiche.**

> Lo, in the high places of the kingdom, where snow lay heavy upon
> the broken battlements, Rowan son of no man knelt before that
> which had been buried in the long age of sorrow.

(Kills: pseudo-archaic vocabulary, *son of no man*, *lo.* The
old-tongue mode lengthens and deepens but does not costume itself
in fake archaism. Cadence over wardrobe.)

**Anti-Exemplar 3 — Modern thriller voice.**

> Rowan's heart hammered as adrenaline flooded his system. He
> couldn't believe what he was seeing. This was huge. This changed
> everything.

(Kills: clinical body-vocabulary, *couldn't believe*, *this changed
everything.* Wrong century, wrong cosmology.)

**Anti-Exemplar 4 — The "AI fantasy paragraph."**

> The intricate tapestry of Vaelgard's history wove together myriad
> threads of triumph and tragedy. Notably, the dragons had been
> instrumental in shaping the political landscape, their absence
> leaving a profound void that resonated through every facet of the
> kingdom's culture.

(Kills: every Tier 1 word in one paragraph. Diagnostic use — what
the writer must *not* drift toward.)

**Anti-Exemplar 5 — Old-tongue mode used as decoration (forbidden).**

> They walked, and the snow came down out of the high places, and
> the wind that had been the breath of older mountains turned
> about them and remembered, and the road they walked had been
> walked before by kings whose names had been forgotten, and the
> cold was a cold that knew them.

(Kills: this is old-tongue mode firing for *no reason*. The
mechanic is not active. Nothing is being remembered. It is
bell-tone for its own sake — exactly the slop the hybrid voice is
built to avoid. Old-tongue mode appears only when the crown's
memory-magic is engaged. Otherwise: cold and bodily.)
